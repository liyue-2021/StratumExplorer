# 工作流节点兼容性测试用例

> 依据：`docs/算法节点兼容性校验说明-20260523(1).docx`  
> 画布节点标题旁橙色角标 = **甲方序号·数据族·形态**（如 `8·DAS·B` 表示 8 号 FFT、DAS 族、频域 B 类）  
> 特殊：`入·数据源`、`0·格式转换`、`解25`（解释）、`展37`（展示）  
> **默认不显示角标**；测试时在工具栏点 **「显示序号角标」**（再点一次为「隐藏序号角标」）。不显示时悬停节点 tooltip 仍有「序号」行。

## 一、测试前准备

1. 关闭旧进程后重新编译运行 `StratumExplorer.exe`。
2. 打开 **工作流** 页，清空画布（工具栏「清空画布」）；需要看角标时打开 **「显示序号角标」**。
3. 记录方式：每项写 **通过 / 失败**，失败时抄状态栏红字原文。
4. 连线操作（任选一种）：
   - **A**：从上游节点 **绿色输出口** 拖到下游节点；
   - **B**：从下游 **黄色输入口** 反向拖到上游。
5. 角标含义：`DAS` / `DTS` / `DSS` = 数据族；`A` = 时域，`B` = 频域，`C` = 特征终点，`耦` = 温度耦合。

## 三、基础连线（必须通过）

### TC-01 数据入口

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | 拖入 **入·数据源**、**0·格式转换** | 显示橙色角标 |
| 2 | 连线：数据输入 → 格式转换 | **成功**，出现连线 |
| 3 | 连线：格式转换 → 数据输入 | **失败**（数据输入无输入口或自动反转为成功） |
| 4 | 连线：数据输入 → **1·深度矫正** | **失败**（须先经格式转换） |

### TC-02 格式转换出口

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | 数据输入 → 格式转换 → **1·深度矫正** | **成功** |
| 2 | 格式转换 → **7·F-K**（跳过 A 直接 B） | **失败** |
